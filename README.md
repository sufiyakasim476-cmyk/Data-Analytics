# Sales Analysis Dashboard

## Overview
This project is an interactive Sales Analysis Dashboard created using Microsoft Power BI. The dashboard helps analyze sales performance, profit trends, regional performance, and category-wise analysis using visual reports and business insights.

---

## Objectives
- Analyze overall sales performance
- Identify top-performing regions and categories
- Track monthly sales trends
- Compare profit and sales across products
- Create interactive and easy-to-understand visualizations

---

## Tools Used
- Microsoft Power BI
- Microsoft Excel / CSV Dataset

---

## Dataset Information
The dataset contains sales-related information such as:
- Order ID
- Order Date
- Product Name
- Category
- Region
- Sales
- Profit
- Quantity

---

## Dashboard Features
- KPI Cards for Total Sales, Profit, and Quantity
- Sales by Region Analysis
- Category-wise Sales Distribution
- Monthly Sales Trend Visualization
- Interactive Filters/Slicers
- Business Insights Section

---

## Key Insights
- West region generated the highest sales.
- Technology category contributed the highest profit.
- Sales trends increased during peak seasonal periods.
- Some categories showed high sales but lower profitability.
